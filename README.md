
# 🌦️ Weather App (API Integration)

A simple and interactive **Weather Application** built using HTML, CSS, and JavaScript. This project demonstrates **API integration** by fetching real-time weather data based on the city entered by the user.



## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Displays temperature in Celsius
* ☁️ Shows weather condition (e.g., clear sky, rain)
* ⚡ Real-time data using API
* 🎯 Simple and beginner-friendly UI



## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Weather API from OpenWeather



## 🔌 API Used

This project uses the **OpenWeather API** to fetch weather data.



## ⚙️ How It Works

1. User enters a city name
2. Application sends a request to the weather API
3. API returns weather data in JSON format
4. Temperature and weather condition are displayed



## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser



## 🔑 API Key Setup

1. Sign up at OpenWeather
2. Generate your API key
3. Replace the placeholder in the code:

```javascript
let apiKey = "YOUR_API_KEY";
```

Delhi
